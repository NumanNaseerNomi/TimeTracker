# ⏱️ TimeTracker - Simple Employee Time Tracking System

A lightweight time tracking application built with PHP, MySQL, and Bootstrap. TimeTracker allows employees to clock in/out and generates downloadable reports for payroll and attendance tracking.

## ✨ Features

### 👤 Employee Features
- **Clock In/Out**: Simple one-click clock in/out functionality
- **View History**: See your own time entries
- **Simple Interface**: Clean and easy-to-use dashboard

### 👨‍💼 Admin Features
- **View All Entries**: See all employees' time records
- **Generate Reports**: Create reports by date range
- **Download Reports**: Export reports as CSV or PDF
- **Employee Management**: Add and manage employees
